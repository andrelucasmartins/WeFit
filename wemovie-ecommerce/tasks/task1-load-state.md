## Task 1: Create the Home Screen Load State

Create a loading screen for the Home Screen app/page.tsx. It should be displayed while data is loading and should be replaced by the Home Screen content once the data is ready. The loading screen should be simple but visually appealing and should clearly indicate that content is loading. Use animations or progress indicators to enhance the user experience during loading.
